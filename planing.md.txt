# Project: Learnova

Learnova is a modern EdTech platform focused on exam preparation, learning content, online courses, and future AI-powered learning experiences.

The initial MVP focuses on Teacher Assistant Exam preparation in Thailand, but the architecture must be designed to support future expansion into:

- Teacher Exams
- Government Exams
- Civil Service Exams
- Professional Certifications
- Online Courses
- Video Learning
- Premium Content
- AI Tutor
- Mock Exams
- Learning Analytics
- Mobile Applications

The platform must be scalable and production-ready from the beginning.

---

# Architecture

Use a decoupled architecture.

Clients:

- Learnova Web (Next.js)
- Learnova Mobile (React Native - future)

Backend:

- Learnova API (NestJS)

Infrastructure:

- PostgreSQL
- Redis (future)
- Cloudflare R2

Architecture Diagram:

Clients
├── Web (Next.js)
└── Mobile (React Native)

↓

NestJS API

↓

Prisma ORM

↓

PostgreSQL

↓

Cloudflare R2

# Design Principles

Learnova is a Content Platform.

It is NOT an ERP system.

Public pages must prioritize:

- SEO
- Content Discovery
- Fast Loading
- Mobile Experience

Public pages must NOT use sidebar layouts.

Public Layout:

- Header
- Search
- Main Content
- Footer

Member Layout:

- Dashboard
- Profile
- History

Admin Layout:

- Sidebar
- Management Pages
